# Mancala Gaming Studio Editor

**Internal tool — for use within Mancala Gaming Studios only.**

A browser-based scene editor for composing, positioning, and inspecting Spine 2D skeletal animations. Designers can arrange objects, set animations and skins, attach child skeletons to placeholder bones, and export the full scene as a `.mancala` project file for developers.

---

## 🌐 Live app

👉 **[Open the editor](https://pantel05.github.io/mancala-studio-editor/)**

> Use **Google Chrome** or **Microsoft Edge** for the best experience (native Save/Open dialogs).

---

## 📖 How to use

1. Open the link above in Chrome or Edge
2. Go to **Project → Import…** and select your Spine files (`.skel` or `.json`, `.atlas`, `.png` textures)
3. Position objects on the canvas, set animations, attach children to placeholder bones
4. Press **⌘S** (Mac) or **Ctrl+S** (Windows) to save your scene as a `.mancala` file
5. Share the `.mancala` file with your team via Google Drive — anyone can open it with **Project → Open…**

For a full feature guide, click the **Help** button in the top bar of the editor.

---

## 💾 Project files (.mancala)

- Self-contained archive — includes all Spine assets + scene layout
- Open by dragging the file onto the canvas, or via **Project → Open…**
- Share via Google Drive, Slack, or any file-sharing tool
